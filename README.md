# Native Spotify Clone
A spotify clone project.

![Project Preview](./gitassets/project-preview.png)

## Technologies
* expo
* react native
* typescript
* styled components

## Getting Started
### installation:
```
git init
https://github.com/eddotbarbosa/native-spotify-clone
npm install
```
### running:
start
```
npm start
```
android
```
npm run android
```
ios
```
npm run ios
```
web
```
npm run web
```
